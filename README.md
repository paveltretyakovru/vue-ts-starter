# Webpack + Vue + TypeScript starter kit

## Instalation

```shell
  git clone https://github.com/ptretyakov/vue-ts-starter.git
  npm install
```

## Starting

```shell
  npm start
```

... and open [localhost:8080](http://localhost:8080)
